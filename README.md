# FastFate

3D 소울라이크 액션 게임입니다.


## 주의사항
이 프로젝트에는 크기 문제로 에셋이 포함되지 않았습니다.
게임 실행을 위해서는 별도의 구글 드라이브 파일을 압축 해제 후 사용해주세요.





### 실행
```bash
# 압축파일 다운로드 후 FastFate.exe 파일 실행
```



## 조작법
```
- **WASD**: 이동
- **Shift**: 달리기
- **마우스**: 카메라 회전
- *Left Click : 공격하기

- **Space**: 구르기(2마나 소비)
- ***Q***: 맨 왼쪽의 카드 버리기
- ***E***: 맨 왼쪽의 카드 사용하기(카드에 해당하는 마나 소비)
- ***R***: 포션 사용하기(2마나 소비)
- **ESC**: 종료
```
